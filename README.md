# 🏠 Home Decor

**Home Decor** is a beginner-level React-based e-commerce website for selling home decoration products.
It allows users to browse decorative items, view product details, and explore beautiful home styling collections.

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/the-abed/home-decor.git

# Go to project directory
cd home-decor

# Install dependencies
npm install

# Run the project
npm start
```

The app will run on:

```
http://localhost:3000
```
